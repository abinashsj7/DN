# Docker Basics

## What is Docker?

Docker is an open-source platform used to develop, package, and run applications inside lightweight containers.

A container includes:
- Application code
- Libraries
- Dependencies
- Runtime environment

This ensures the application runs consistently on any machine.

---

## Why Docker?

- Platform independent
- Faster deployment
- Lightweight
- Easy scalability
- Better resource utilization

---

## Benefits

- Consistent environments
- Faster software delivery
- Easy application deployment
- Simplified dependency management

---

## Real-world Example

Without Docker:
"It works on my computer."

With Docker:
"It works the same on every computer."
