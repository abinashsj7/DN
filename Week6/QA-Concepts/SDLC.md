# SDLC (Software Development Life Cycle)

## Definition

Software Development Life Cycle (SDLC) is a structured process used to develop high-quality software from planning to maintenance.

## Phases

1. Requirement Analysis
2. Planning
3. Design
4. Development
5. Testing
6. Deployment
7. Maintenance

## Diagram

Requirements
      ↓
Planning
      ↓
Design
      ↓
Development
      ↓
Testing
      ↓
Deployment
      ↓
Maintenance

## Advantages

- Better planning
- High quality software
- Reduced cost
- Easier project management
