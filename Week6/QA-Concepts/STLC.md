# STLC (Software Testing Life Cycle)

## Definition

STLC is the process followed by the testing team to ensure software quality.

## Phases

1. Requirement Analysis
2. Test Planning
3. Test Case Development
4. Test Environment Setup
5. Test Execution
6. Defect Reporting
7. Test Closure

## Diagram

Requirement Analysis
        ↓
Test Planning
        ↓
Test Case Development
        ↓
Environment Setup
        ↓
Test Execution
        ↓
Bug Reporting
        ↓
Test Closure

## Goal

Deliver a bug-free application with high quality.
