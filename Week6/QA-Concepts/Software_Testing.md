# Software Testing

## What is Software Testing?

Software Testing is the process of checking whether a software application works as expected and identifying defects before it is released to users.

## Objectives

- Find defects
- Ensure quality
- Improve reliability
- Verify requirements
- Validate functionality

## Why is Testing Important?

- Improves software quality
- Reduces development cost
- Enhances customer satisfaction
- Prevents failures in production

## Example

Suppose an ATM should dispense cash after entering the correct PIN.

Expected Result:
Cash is dispensed.

Actual Result:
Cash is not dispensed.

This is called a defect (bug).
