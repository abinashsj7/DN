# Git vs GitHub

## Git

Git is a Distributed Version Control System (DVCS) used to track changes in source code.

### Features
- Tracks code changes
- Works locally
- Supports branching and merging
- Open Source

---

## GitHub

GitHub is a cloud-based platform used to host Git repositories and collaborate with developers.

### Features
- Stores repositories online
- Team collaboration
- Pull Requests
- Issue Tracking
- GitHub Actions (CI/CD)

---

## Difference

| Git | GitHub |
|------|---------|
| Version Control System | Cloud Hosting Platform |
| Works locally | Works online |
| Tracks code | Stores Git repositories |
| Command-line tool | Web application |

## Example

Git → Saves versions on your computer.

GitHub → Stores those versions online so others can access them.
