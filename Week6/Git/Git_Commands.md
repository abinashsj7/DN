# Git Commands

## Initialize a Repository

```bash
git init
```

Creates a new Git repository.

---

## Check Repository Status

```bash
git status
```

Shows modified and untracked files.

---

## Add Files

```bash
git add .
```

Stages all files.

```bash
git add filename.py
```

Stages a specific file.

---

## Commit Changes

```bash
git commit -m "Initial commit"
```

Saves the staged changes.

---

## View Commit History

```bash
git log
```

Displays commit history.

---

## Clone Repository

```bash
git clone <repository-url>
```

Downloads a remote repository.

---

## Push Changes

```bash
git push origin main
```

Uploads commits to GitHub.

---

## Pull Changes

```bash
git pull origin main
```

Downloads and merges changes.

---

## Create Branch

```bash
git branch feature
```

---

## Switch Branch

```bash
git checkout feature
```

---

## Merge Branch

```bash
git merge feature
```
