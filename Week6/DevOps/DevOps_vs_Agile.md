# DevOps vs Agile

| Agile | DevOps |
|--------|---------|
| Focuses on software development | Focuses on development and operations |
| Delivers software in iterations | Delivers software continuously |
| Developers and testers collaborate | Developers, testers, and operations collaborate |
| Improves development speed | Improves development and deployment speed |

---

## Relationship

Agile improves the software development process.

DevOps extends Agile by automating building, testing, deployment, and monitoring.
