## Description

### Plan
Gather requirements.

### Develop
Write source code.

### Build
Compile and package the application.

### Test
Run automated and manual tests.

### Release
Prepare the software for deployment.

### Deploy
Deploy to production.

### Operate
Maintain the application.

### Monitor
Track performance and fix issues.
