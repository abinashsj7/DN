# Continuous Delivery & Continuous Deployment

## Continuous Delivery

Continuous Delivery ensures that software is always ready for release.

After testing, deployment requires **manual approval**.

Workflow:

Code
 ↓
Build
 ↓
Test
 ↓
Manual Approval
 ↓
Production

---

## Continuous Deployment

Continuous Deployment automatically deploys every successful change to production without manual intervention.

Workflow:

Code
 ↓
Build
 ↓
Test
 ↓
Automatic Deployment
 ↓
Production

---

## Benefits

- Faster releases
- Reduced manual effort
- Reliable deployments
- Better customer satisfaction
