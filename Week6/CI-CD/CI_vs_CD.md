# CI vs CD

| Continuous Integration (CI) | Continuous Delivery (CD) |
|------------------------------|---------------------------|
| Integrates code frequently | Delivers software for release |
| Focuses on building and testing | Focuses on deployment |
| Detects bugs early | Ensures release readiness |
| Developers use it daily | Release teams use it before production |

---

## Continuous Delivery vs Continuous Deployment

| Continuous Delivery | Continuous Deployment |
|---------------------|----------------------|
| Manual approval required | Fully automatic deployment |
| Human intervention | No human intervention |
| Lower deployment frequency | Higher deployment frequency |

---

## Interview Tip

CI = Build + Test

CD = Deploy
