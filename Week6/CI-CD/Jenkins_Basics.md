# Jenkins Basics

## What is Jenkins?

Jenkins is an open-source automation server used to implement Continuous Integration (CI) and Continuous Delivery (CD).

## Features

- Automates builds
- Runs automated tests
- Deploys applications
- Supports plugins
- Integrates with Git and Docker
